# Umbra Yurei Homebrew Tap

This tap distributes [Cruma](https://cruma.io) tunnel agent binaries through Homebrew.

```bash
brew tap umbra-yurei/brew
```

---

## Installing Cruma

### macOS — app bundle (GUI + CLI)

Installs `Cruma.app` to `/Applications` and symlinks the `cruma` binary into your PATH.

```bash
brew install --cask umbra-yurei/brew/cruma
```

After installation, both the menu-bar app and the `cruma` CLI command are available.

### Linux — CLI binary (static, musl)

Installs a statically linked `cruma` binary. No GUI.

```bash
brew install umbra-yurei/brew/cruma
```

---

## Preview builds

Preview builds track the latest beta/release-candidate. They install alongside
stable using separate names:

- macOS app: `Cruma Preview.app`
- CLI command: `cruma-preview`

### macOS (preview app + CLI)

```bash
brew install --cask umbra-yurei/brew/cruma-preview
```

This installs `Cruma Preview.app` and symlinks `cruma-preview` into your PATH.

### Linux (preview CLI binary)

```bash
brew install umbra-yurei/brew/cruma-preview
```

This installs the `cruma-preview` CLI binary.

---

## Upgrading

```bash
# Stable
brew upgrade umbra-yurei/brew/cruma          # formula
brew upgrade --cask umbra-yurei/brew/cruma   # cask

# Preview
brew upgrade umbra-yurei/brew/cruma-preview
brew upgrade --cask umbra-yurei/brew/cruma-preview
```

## Releasing a new version

Formulas and casks are generated by `g.sh`. Artifacts must be uploaded to
`files.cruma.io` under `tunnel-agent/v<version>/` before running it.

**Expected artifact layout on files.cruma.io:**

```
tunnel-agent/v<version>/
  aarch64-apple-darwin/cruma.dmg
  x86_64-unknown-linux-gnu/cruma
  aarch64-unknown-linux-musl/cruma
```

**Generate stable release:**

```bash
./g.sh 0.3.0
```

**Generate preview release:**

```bash
./g.sh 0.3.0-beta.3 --preview
```

Both commands download the artifacts, compute SHA256 hashes, and write:

| File | Description |
|---|---|
| `Formula/cruma.rb` | Linux CLI formula (stable) |
| `Casks/cruma.rb` | macOS cask (stable) |
| `Formula/cruma-preview.rb` | Linux CLI formula (preview) |
| `Casks/cruma-preview.rb` | macOS cask (preview) |

After running, review the generated files and commit:

```bash
git add Formula/ Casks/
git commit -m "Release v0.3.0"
git push
```
